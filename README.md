## To Run Project in your Local PC follow the following steps:

### Clone My GitHub Repository from:
```
https://github.com/Joyontokarmakar/xyz-engine.git
```

### Installation

``` bash
$ npm install
```

or

``` bash
$ yarn
```

### Run Your Project

``` bash
$ npm run dev 
```

or

``` bash
$ yarn dev
```
